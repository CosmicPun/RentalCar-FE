Design a clean and minimal UI Flow for a Rental Car Booking web application using a yellow and black color theme. Primary color is #FFD600 (yellow), background is #FFFFFF, accent/text is #111111 (near black).  Create the following screens as a UI Flow with arrows connecting the user journey:  

1. **Register Page** — Form with fields: Name, Telephone, Email, Password, Confirm Password. Include a Register button and a link to Login.  

2. **Login Page** — Form with Email and Password fields. Login button and link to Register.  

3. **Home / Car Provider List Page** — Grid or card list of rental car providers. Each card shows: Provider Name, Address, Telephone Number. Include a top navbar with logo and logout button.  

4. **Create Booking Page** — Form to book a car: select rental car provider (dropdown), select date (date picker), with a Submit Booking button. Show a validation warning if user tries to book more than 3 cars.  

5. **My Bookings Page** — Table or card list showing the user's own bookings with columns: Provider Name, Date, Actions (Edit / Delete buttons).  

6. **Edit Booking Page** — Pre-filled form to update an existing booking (provider, date). Save and Cancel buttons.  

7. **Admin Dashboard Page** — Table showing ALL bookings from all users: User Name, Provider, Date, Actions (Edit / Delete).  Connect all screens with flow arrows showing the user journey. Label each arrow with the action (e.g., "Click Login", "Submit Booking", "Click Edit").  Style: Clean and minimal. Use yellow (#FFD600) for primary buttons and highlights. Black (#111111) for text and navbar. White background. Rounded corners on cards and buttons. Use a modern sans-serif font like Inter or Poppins.
